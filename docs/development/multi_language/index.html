    多言語対応 #

    COCOAの文字列リソースはCovid19Radar/Covid19Radar/Resources以下にあるAppResources.*.resxファイルにあります。

    多言語対応をする際、たとえば日本語の文字列リソースを変更するときは AppResources.ja.resx を編集します。

    以前は多言語対応にMultilingual App Toolkitを利用していましたが、現在は廃止されています。
